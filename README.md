# RH Dashboard - La Pratique Electronique

## 📊 Tableau de Bord RH

Application interactive pour le suivi des indicateurs RH.

### 📈 Indicateurs disponibles :

- **Effectifs** : Suivi des effectifs par service, catégorie, sexe
- **Mouvements** : Entrées, sorties, turnover, motifs de départ
- **Talents** : Promotions, délai moyen de promotion
- **Administration** : Enquêtes, entretiens, sanctions, absentéisme
- **KPIs** : Qualité recrutement, fuite des compétences, score de risque
- **Alertes** : Détection automatique des risques RH

### 🚀 Technologies utilisées
- Streamlit
- Pandas
- Plotly
- Python
